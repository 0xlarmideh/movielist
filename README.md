## To run the project
- Clone this repo by entering `git clone https://github.com/0xlarmideh/movielist.git`
- cd into the project folder
- run `npm install`
- run `npm run dev` or `yarn run dev` to start the local server

## Movielist App

- Add to watchlist
- Remove from watchlist
- Toggle watch status
- Rate movies

## Stack

- ChakraUI
- Redux Toolkit
